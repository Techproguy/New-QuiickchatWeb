# Quiickchat Frontend

A modern chat application built with Next.js 15, featuring QR code authentication, real-time messaging, and translation capabilities.

## 🚀 Features

- **QR Code Authentication** - Secure login via mobile app scanning
- **Real-time Messaging** - Instant chat with delivery status
- **Translation Support** - Translate messages between languages
- **Voice & Video Calls** - Full communication suite
- **Responsive Design** - Works on all devices
- **Modern UI** - Clean, intuitive interface

## 🛠️ Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS
- **UI Components**: Radix UI
- **Icons**: Lucide React
- **TypeScript**: Full type safety
- **Deployment**: Vercel

## 📦 Installation

\`\`\`bash
# Clone the repository
git clone https://github.com/sarmad0324/quick-chat-web.git

# Install dependencies
pnpm install

# Run development server
pnpm dev
\`\`\`

## 🌐 Deployment

This project is optimized for Vercel deployment:

\`\`\`bash
# Build for production
pnpm build

# Start production server
pnpm start
\`\`\`

## 📱 Pages

- `/` - Home (redirects to login/chat)
- `/login` - QR code authentication
- `/chat` - Main chat interface
- `/calls` - Voice/video calls
- `/recent` - Recent activity
- `/settings` - User preferences
